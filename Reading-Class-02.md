# Class Basic Syntax
https://javascript.info/class

* a class is a kind of function
* basic syntax and usage:
    ```js
    class User {

  constructor(name) {
    this.name = name;
  }

  sayHi() {
    alert(this.name);
  }

    }

    // Usage:
    let user = new User("John");
    user.sayHi();
    ```
